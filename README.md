# Malyna
Bot do naszego IRC: freenode.net #mzforum

## Wymagania
```
$ apt-get install python-pip
$ apt-get install xclip
```
## Instalacja
1. Pobrać kod Malyny

  ```
  $ git clone https://github.com/CodersCommunity/Malyna.git
  $ cd Malyna
  ```
2. Instalacja zależności

  ```
  $ pip install -r requirements.txt
  ```

### Uruchomienie
```
$ python malyna.py
```
